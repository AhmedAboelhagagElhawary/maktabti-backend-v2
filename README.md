# مكتبتي - Maktabti
## Intelligent Library Management System Backend

### نظام إدارة مكتبة ذكي

**التقنيات المستخدمة:**
- Node.js + Express.js
- MongoDB Atlas
- JWT Authentication
- Railway Hosting

### التثبيت

\`\`\`bash
npm install
\`\`\`

### البدء

\`\`\`bash
npm run dev
\`\`\`

### المتطلبات

يجب إنشاء ملف `.env`:

\`\`\`
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
NODE_ENV=development
\`\`\`
